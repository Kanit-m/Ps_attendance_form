# คู่มือการใช้งานระบบรายงานการมาเรียน (Attendance Report System Usage Guide)

ระบบนี้ถูกออกแบบมาให้เป็นไฟล์เดียว (Single File Application) เพื่อความสะดวกในการใช้งานและการติดตั้ง

## 1. การใช้งานแบบออฟไลน์ (Local Use)
วิธีที่ง่ายที่สุดในการใช้งานคือการเปิดไฟล์บนเครื่องคอมพิวเตอร์ของคุณโดยตรง

1.  ไปที่โฟลเดอร์ที่เก็บไฟล์ `preview.html` (เช่น `c:\Webapp\form\`)
2.  ดับเบิลคลิกที่ไฟล์ `preview.html`
3.  ไฟล์จะถูกเปิดผ่านเว็บเบราว์เซอร์ (แนะนำ Google Chrome หรือ Microsoft Edge)
4.  **ข้อควรระวัง:** เครื่องคอมพิวเตอร์ต้องเชื่อมต่ออินเทอร์เน็ต เพื่อให้สามารถโหลดไลบรารี (React, Firebase) และเชื่อมต่อฐานข้อมูลได้

## 2. การนำไปใช้บนเซิร์ฟเวอร์ (Hosting)
หากต้องการให้ครูท่านอื่นหรือผู้อำนวยการเข้าดูได้จากเครื่องอื่น คุณต้องนำไฟล์นี้ไปวางบนเว็บเซิร์ฟเวอร์

### ทางเลือกที่ 1: Firebase Hosting (แนะนำ)
เนื่องจากเราใช้ Firebase อยู่แล้ว การใช้ Firebase Hosting จะสะดวกที่สุด
1.  ติดตั้ง Firebase CLI: `npm install -g firebase-tools`
2.  ล็อกอิน: `firebase login`
3.  รันคำสั่ง `firebase init` ในโฟลเดอร์โปรเจกต์
    *   เลือก **Hosting**
    *   เลือกโปรเจกต์เดิม (`tester010-1a27e`)
    *   ตั้งค่า public directory เป็นโฟลเดอร์ที่เก็บไฟล์ (หรือย้าย `preview.html` ไปที่โฟลเดอร์ `public` แล้วเปลี่ยนชื่อเป็น `index.html`)
4.  รันคำสั่ง `firebase deploy`

### ทางเลือกที่ 2: เว็บโฮสติ้งทั่วไป
คุณสามารถอัปโหลดไฟล์ `preview.html` ไปยังเว็บโฮสติ้งใดก็ได้ที่รองรับ HTML static files (เช่น GitHub Pages, Netlify, Vercel หรือโฮสติ้งของโรงเรียน)

## 3. การเตรียมไฟล์ CSV สำหรับนำเข้า
เพื่อให้การนำเข้าข้อมูลถูกต้อง ไฟล์ CSV ควรมีการตั้งค่าดังนี้:
*   **Encoding:** ควรบันทึกเป็น **UTF-8** หรือถ้าบันทึกจาก Excel โดยตรงให้เลือกตัวเลือก **TIS-620 (Excel)** ในหน้าเว็บตอนอัปโหลด
*   **Columns:** ตรวจสอบลำดับคอลัมน์ในเมนู "ตั้งค่าคอลัมน์" ให้ตรงกับไฟล์ของคุณ

## 4. การพิมพ์และบันทึกเป็น PDF
*   **พิมพ์รายงาน:** คลิกปุ่มสีเขียว "พิมพ์รายงาน (Print)" ระบบจะเรียกหน้าต่างสั่งพิมพ์ของเบราว์เซอร์ขึ้นมา
*   **บันทึก PDF:** คลิกปุ่มสีแดง "ดาวน์โหลด PDF" ระบบจะสร้างไฟล์ PDF ให้ดาวน์โหลดทันที
